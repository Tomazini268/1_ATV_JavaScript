 Primeira Atividade em JavaScript – Programa de Boas-Vindas
 
Descrição

Projeto introdutório em JavaScript que solicita o nome do usuário por meio de uma caixa de diálogo e exibe uma mensagem de boas-vindas personalizada na página.

 Objetivo

Praticar a integração de JavaScript com HTML

Utilizar entrada de dados com window.prompt()

Exibir informações usando document.write()

 Tecnologias

HTML

JavaScript

Funcionamento

O navegador solicita o nome do usuário

O nome digitado é armazenado em uma variável

A página exibe a mensagem: “Olá [Nome] Seja Bem Vindo!”
